## Variant: Premium Dark 質感卡

### Design stance
高級、沉浸、精品感 — 深色玻璃效果讓頁面像一張 VIP 邀請卡

### Key choices
- 深色背景 (#0b0e14) + 玻璃卡片 (backdrop-filter)，創造高級感
- 發光光暈在頭像周圍旋轉，動態吸引目光
- 三個特色標籤（素食饗宴/最新消息/活動通知）讓朋友知道加入後能得到什麼
- 漸層按鈕 + 陰影，按壓回饋明確
- 所有顏色低飽和度，只有 LINE 綠跳出來引導點擊
- 邊框發光效果，卡片像浮在半空中

### Trade-offs
- 強項：視覺高級、品牌印象深刻、適合推薦給講究的朋友
- 弱項：暗色背景在強光下可讀性較低
